# AI Engineer Social Impact Experiment

이 프로젝트는 엔지니어의 기술적 결정이 노동 환경에 미치는 영향을 실험하기 위한 웹앱입니다.

## 기능
- **Step 1:** 효율성 중심의 AI 콜센터 기능 설계
- **Step 2:** 상담원 인터뷰를 통한 딜레마(Intervention) 제공
- **Step 3:** 사회적 가치를 고려한 기능 재설계
- **Step 4:** Before/After 결과 비교

## 실행 방법
1. `pip install -r requirements.txt`
2. `streamlit run app.py`
